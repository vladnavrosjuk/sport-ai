# Команда: /generate-nutrition

Генерация персонализированного плана питания.

## Описание

Создаёт план питания на основе данных из профиля пользователя.
Рассчитывает калории, макронутриенты и предлагает примерное меню.

## Требования

- Должен существовать файл `.cursor/project/user_profile.md`
- Если профиля нет, будет предложено запустить `/fitness-init`

## Процесс генерации

1. **Анализ профиля** — загрузка данных пользователя
2. **Расчёт BMR** — базовый метаболизм
3. **Расчёт TDEE** — общий расход энергии
4. **Целевые калории** — с учётом цели (набор/похудение)
5. **Расчёт макросов** — белки, жиры, углеводы
6. **Распределение** — по приёмам пищи
7. **Генерация плана** — создание файла

## Результат

Создаётся файл `docs/user/nutrition_plan.md` содержащий:
- Целевые калории и макросы
- Расписание приёмов пищи
- Примерное меню на день
- Список продуктов на неделю
- Правила корректировки

## Параметры

Можно уточнить при запуске:
- **Количество приёмов пищи** — 3/4/5/6
- **Время тренировки** — для оптимизации тайминга
- **Бюджет** — для подбора продуктов

## Связанные правила

- [meal_planning.mdc](mdc:../rules/nutrition/meal_planning.mdc)
- [calorie_calculator.mdc](mdc:../rules/nutrition/calorie_calculator.mdc)

## Примечания

- План создаётся с учётом пищевых ограничений из профиля
- Для записи веса используйте `/log-weight`
- Для корректировки калорий используйте `/adjust-calories`
