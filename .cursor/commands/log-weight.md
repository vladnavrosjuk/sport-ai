# Команда: /log-weight

Запись текущего веса тела.

## Описание

Записывает вес, анализирует динамику и предлагает корректировки питания при необходимости.

## Рекомендации по взвешиванию

- Утром, натощак
- После туалета
- В одинаковых условиях (одежда/без)
- Минимум 3 раза в неделю

## Процесс

1. **Ввод веса** — в килограммах
2. **Запись в лог** — с датой и временем
3. **Анализ** — сравнение с целью и прошлыми данными
4. **Рекомендации** — корректировки питания

## Результат

- Запись добавляется в `docs/user/weight_log.md`
- Обновляется вес в профиле
- При необходимости предлагается `/adjust-calories`

## Автоматический анализ

При записи агент анализирует:
- Тренд за последние 7 дней
- Средненедельный вес
- Соответствие целевому темпу изменения

## Связанные команды

- `/adjust-calories` — корректировка калорий
- `/weekly-review` — еженедельный обзор

## Примечания

- Вес может колебаться ±1-2 кг в течение дня
- Анализируйте средненедельный вес, а не дневные значения
- После начала тренировок возможна задержка воды
