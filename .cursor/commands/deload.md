# Команда: /deload

Планирование разгрузочной недели.

## Описание

Генерирует программу разгрузочной недели (deload) для восстановления и профилактики перетренированности.

## Когда использовать

### Запланированный deload
- Каждые 4-6 недель тренировок
- После завершения мезоцикла
- Перед тестированием максимумов

### По состоянию
- Снижение силы 2+ тренировки подряд
- Постоянная усталость
- Нарушения сна
- Боли в суставах
- Снижение мотивации

## Методы deload

### 1. Снижение объёма (рекомендуется)
```
Подходы: 50-60% от обычного
Интенсивность: 100%
Пример: 4×5 → 2×5 с тем же весом
```

### 2. Снижение интенсивности
```
Подходы: 100%
Интенсивность: 60-70%
Пример: 100 кг → 65 кг
```

### 3. Комбинированный
```
Подходы: 60%
Интенсивность: 70%
Самый консервативный
```

### 4. Полный отдых
```
Нет тренировок 5-7 дней
Только при сильной усталости
```

## Результат

Генерируется облегчённая программа на неделю с:
- Сокращённым количеством упражнений
- Сниженным объёмом или интенсивностью
- Рекомендациями по восстановлению

## Что делать во время deload

✅ Рекомендуется:
- Лёгкое кардио 20-30 мин
- Растяжка и мобильность
- Полноценный сон
- Достаточное питание

❌ Не рекомендуется:
- Интенсивные тренировки
- Тестирование максимумов
- Резкое снижение калорий

## Примечания

- После deload обычно наблюдается "отскок" силы
- Питание — на уровне поддержания
- Это не "неделя без тренировок"
