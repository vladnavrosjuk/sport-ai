# Sport AI Workplace

> **Персонализированный AI-тренер для силовых тренировок**
> 
> Cursor-шаблон, который генерирует индивидуальные программы тренировок и питания на основе ваших параметров, целей и текущего уровня подготовки.

---

## Быстрый старт

### 1. Инициализация профиля

Откройте Cursor и выполните команду:

```
/fitness-init
```

AI-агент проведёт вас через опрос:
- Базовые параметры (рост, вес, возраст, пол)
- Текущие силовые показатели
- Тренировочный опыт и травмы
- Цели и предпочтения
- Диетические ограничения

### 2. Генерация программы

После инициализации доступны команды:

| Команда | Описание |
|---------|----------|
| `/generate-workout` | Создать программу тренировок |
| `/generate-nutrition` | Создать план питания |
| `/generate-full` | Полная программа: тренировки + питание |
| `/update-progress` | Обновить прогресс и скорректировать программу |
| `/next-cycle` | Перейти к следующему мезоциклу |

### 3. Отслеживание прогресса

```
/log-workout
```

Записывайте результаты тренировок, и AI будет автоматически корректировать нагрузку.

## Возможности

### Тренировки
- **Периодизация** — линейная, волновая, блочная
- **Силовые программы** — пауэрлифтинг, бодибилдинг, общая сила
- **Автоподстройка** — прогрессия нагрузки на основе ваших результатов
- **Учёт восстановления** — деload-недели, RPE-контроль

### Питание
- **Расчёт КБЖУ** — на основе целей и активности
- **План питания** — с учётом предпочтений и ограничений
- **Тайминг нутриентов** — распределение по приёмам пищи
- **Корректировка** — автоматическая подстройка при изменении веса

### Прогресс
- **Трекинг силовых** — PR в базовых упражнениях
- **Трекинг веса** — динамика массы тела
- **Аналитика** — графики и рекомендации
- **Предсказания** — прогноз достижения целей

## Конфигурация

Профиль пользователя хранится в `.cursor/project/user_profile.md` и содержит:

- Антропометрические данные
- Силовые показатели
- Историю тренировок
- Текущую программу
- План питания
