# Начало работы с Sport AI Workplace

Добро пожаловать! Этот гайд поможет вам настроить персонального AI-тренера.

## Быстрый старт (5 минут)

### Шаг 1: Инициализация профиля

Откройте Cursor и выполните команду:

```
/fitness-init
```

AI-агент проведёт вас через серию вопросов:

1. **Базовые данные** — имя, возраст, пол, рост, вес
2. **Тренировочный опыт** — стаж, силовые показатели, травмы
3. **Цели** — набор массы, похудение, сила
4. **Образ жизни** — расписание, оборудование, питание

### Шаг 2: Генерация программы

После создания профиля выполните:

```
/generate-full
```

Это создаст:
- Программу тренировок на 4-6 недель
- План питания с расчётом калорий и макросов

Или по отдельности:
- `/generate-workout` — только тренировки
- `/generate-nutrition` — только питание

### Шаг 3: Начните тренироваться!

Используйте сгенерированную программу из `docs/user/current_program.md`.

---

## Ежедневное использование

### После каждой тренировки

```
/log-workout
```

Запишите результаты — это поможет AI отслеживать прогресс и корректировать программу.

### Взвешивание (2-3 раза в неделю)

```
/log-weight
```

Записывайте вес утром натощак для точного отслеживания.

### Еженедельно

```
/weekly-review
```

Получите анализ прогресса и рекомендации.

---

## Структура программы

### Тренировочный мезоцикл

Программа строится на мезоциклах по 4-6 недель:

```
Неделя 1-2: Накопление (умеренная нагрузка)
Неделя 3-4: Интенсификация (тяжёлые веса)
Неделя 5:   Deload (разгрузка)
```

### Переход к следующему циклу

```
/next-cycle
```

Агент проанализирует результаты и создаст новую программу.

---

## Разгрузочная неделя (Deload)

Каждые 4-6 недель нужна разгрузка:

```
/deload
```

Признаки, что нужен deload:
- Падение силы 2+ тренировки
- Постоянная усталость
- Боли в суставах
- Плохой сон

---

## Корректировка питания

Если вес не меняется в нужном направлении:

```
/adjust-calories
```

Агент предложит увеличить или уменьшить калорийность.

## FAQ

### Я новичок, с чего начать?

1. Запустите `/fitness-init`
2. Честно ответьте на вопросы (даже если показатели скромные)
3. Агент создаст программу для вашего уровня

### Как часто обновлять программу?

Не чаще раза в 4-6 недель. Стабильность программы важна для прогресса.

### Что делать, если пропустил тренировку?

Просто продолжите с того места, где остановились. Не пытайтесь "догнать".

### Как узнать свой 1RM?

Если не знаете точно — агент рассчитает по вашему рабочему весу и количеству повторений.

### Питание обязательно отслеживать?

Для оптимального прогресса — да. Но можно начать только с тренировок и добавить питание позже.
