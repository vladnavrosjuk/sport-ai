# Команда: /generate-workout

Генерация персонализированной программы тренировок.

## Описание

Создаёт программу тренировок на основе данных из профиля пользователя.
Учитывает цели, уровень подготовки, доступное время и оборудование.

## Требования

- Должен существовать файл `.cursor/project/user_profile.md`
- Если профиля нет, будет предложено запустить `/fitness-init`

## Процесс генерации

1. **Анализ профиля** — загрузка данных пользователя
2. **Выбор сплита** — определение оптимального разбиения
3. **Подбор упражнений** — с учётом оборудования и ограничений
4. **Расчёт объёма** — количество подходов и повторений
5. **Расчёт интенсивности** — рабочие веса на основе 1RM
6. **Генерация программы** — создание файла с программой

## Результат

Создаётся файл `docs/user/current_program.md` содержащий:
- Расписание тренировок
- Список упражнений с подходами/повторами/весами
- Рекомендации по отдыху
- Правила прогрессии
- Заметки по технике

## Параметры

Можно уточнить при запуске:
- **Тип программы** — сила/гипертрофия/смешанный
- **Акцент** — на какие группы мышц сделать упор
- **Продолжительность цикла** — 4/6/8 недель

## Связанные правила

## Примечания

- Программа создаётся на один мезоцикл (4-6 недель)
- После завершения цикла используйте `/next-cycle`
- Для записи результатов используйте `/log-workout`
