# Команда: /log-workout

Запись результатов тренировки в лог.

## Описание

Записывает фактические результаты тренировки, сравнивает с планом и предлагает корректировки.

## Требования

- Должна существовать программа тренировок (`docs/user/current_program.md`)

## Процесс

1. **Выбор тренировки** — из текущей программы
2. **Ввод результатов** — подходы, повторения, вес, RPE
3. **Сравнение с планом** — процент выполнения
4. **Анализ** — прогресс относительно прошлых тренировок
5. **Рекомендации** — корректировки весов/объёма

## Формат записи

```
Дата: [сегодня]
Тренировка: [название]

| Упражнение | План | Факт | RPE |
|------------|------|------|-----|
| Приседания | 100×5×4 | 100×5,5,5,4 | 8 |
...
```

## Результат

- Запись добавляется в `docs/user/workout_log.md`
- Обновляются силовые показатели (если новый PR)
- Предлагаются корректировки для следующей тренировки

## Связанные правила

- [progress_tracking.mdc](mdc:../rules/training/progress_tracking.mdc)

## Примечания

- Записывайте тренировку сразу после выполнения
- RPE (Rating of Perceived Exertion) — шкала 1-10
- При пропуске тренировки можно отметить причину
