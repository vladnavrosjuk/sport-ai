# Команда: /next-cycle

Переход к следующему тренировочному мезоциклу.

## Описание

Завершает текущий мезоцикл, анализирует результаты и генерирует следующий цикл с учётом прогресса.

## Когда использовать

- После завершения текущего мезоцикла (4-6 недель)
- После deload-недели
- При необходимости сменить фокус программы

## Процесс

1. **Анализ завершённого цикла**
   - Прогресс в силовых
   - Достижение целей
   - Проблемные области

2. **Обновление показателей**
   - Новые 1RM
   - Обновлённый вес тела
   - Корректировка уровня подготовки

3. **Выбор следующего цикла**
   - Тип (сила/гипертрофия/выносливость)
   - Продолжительность
   - Акценты

4. **Генерация новой программы**
   - Новые рабочие веса
   - Возможно новые упражнения
   - Обновлённый объём

## Параметры

Можно указать:
- **Тип цикла** — сила/гипертрофия/поддержание
- **Акцент** — слабые места для проработки
- **Продолжительность** — 4/5/6 недель

## Результат

- Архивируется старая программа
- Создаётся новая в `docs/user/current_program.md`
- Обновляется профиль

## Связанные правила

- [program_generation.mdc](mdc:../rules/training/program_generation.mdc)

## Примечания

- Рекомендуется делать deload перед новым циклом
- Не меняйте программу чаще чем раз в 4 недели
- Следуйте принципу прогрессии
