# Команда: /show-progress

Показать текущий прогресс и статистику.

## Описание

Отображает сводку прогресса: силовые показатели, вес тела, выполнение программы.

## Что показывается

### Силовые показатели
- Текущие 1RM
- Прогресс за последний месяц
- Визуализация относительно целей

### Вес тела
- Текущий вес и тренд
- График динамики
- Прогноз достижения цели

### Тренировки
- Выполненные за период
- Процент соблюдения программы
- Пропущенные тренировки

### Питание
- Средняя калорийность
- Соблюдение плана
- Корреляция с весом

## Параметры

- **Период** — неделя/месяц/всё время
- **Детализация** — краткий/подробный отчёт

## Пример вывода

```
📊 ПРОГРЕСС

Силовые (1RM):
┌─────────┬────────┬────────┬─────────┐
│ Упражн. │ Было │ Сейчас │ Цель │
├─────────┼────────┼────────┼─────────┤
│ Присед │ 100 кг │ 120 кг │ 150 кг │
│ ████████░░░░ 80% │
└─────────┴────────┴────────┴─────────┘

Вес тела: 85 кг → 87 кг (+2 кг)
Тренировок: 12/16 (75%)
```

## Связанные команды

- `/weekly-review` — еженедельный обзор
- `/monthly-review` — ежемесячный обзор
